# 智能巡检小车 - 全局详细接线表 (根据软件代码配置解析)

> **基准代码版本**：基于标准的 STM32F407ZGT6 工程 
> **控制策略说明**：使用了 4 个独立的 TB6612FNG 驱动模块，每个模块对应控制 1 个带编码器的 N20 电机。为保持一致性，统一使用驱动板上的 **A 通道 (PWMA, AIN1, AIN2, AO1, AO2)** 驱动单个电机，B 通道全部悬空不用。

---

## 1. 电源与共地基础（全车最关键）
所有模块的 `GND` 必须与 STM32单片机板子上的 `GND`、电池的负极连通。
- **单片机板 5V**：供给 OpenMV、SYN6658 语音、红外等外设。
- **单片机板 3.3V**：供给 MPU6050、4个TB6612的芯片逻辑(VCC)、N20编码器供电。
- **电池正极 (7V~12V)**：单独拉给 4 个 TB6612 驱动板的 `VM` 引脚，提供暴力输出。

---

## 2. N20 减速电机 (带编码器 6 根线定义)
N20尾部排线一般从边缘到边缘共六个接口，对于4个电机，它们的线序和接法如下：

| 线序序号 (通常丝印) | 引脚含义说明 | 左前(电机1) 接法 | 右前(电机2) 接法 | 左后(电机3) 接法 | 右后(电机4) 接法 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **线1 (红/粗)** | `M+` (电机供电) | 驱动模块1的 `AO1` | 驱动模块2的 `AO1` | 驱动模块3的 `AO1` | 驱动模块4的 `AO1` |
| **线2 (黑/细)** | `VCC` (编码电源) | 3.3V 或 5V | 3.3V 或 5V | 空（代码未开启） | 空（代码未开启） |
| **线3 (黄/细)** | `A相` (编码脉冲)| 单片机 `PA0` | 单片机 `PA6` | 空（代码未开启） | 空（代码未开启） |
| **线4 (绿/细)** | `B相` (编码脉冲)| 单片机 `PA1` | 单片机 `PA7` | 空（代码未开启） | 空（代码未开启） |
| **线5 (蓝/细)** | `GND` (编码地)  | 单片机 GND | 单片机 GND | 空（代码未开启） | 空（代码未开启） |
| **线6 (白/粗)** | `M-` (电机供电) | 驱动模块1的 `AO2` | 驱动模块2的 `AO2` | 驱动模块3的 `AO2` | 驱动模块4的 `AO2` |

> *注：只有左前和右前在代码中配置了 TIM2 和 TIM3 用于捕获编码器数据。后两轮作为随动处理，暂不需要测速。如果任何一个轮子向后倒转，请将对应的 `M+` 和 `M-` (AO1和AO2) 对调即可！*

## 3. 四块 TB6612FNG 驱动板详细引脚 (包含不接的引脚)

每一个 TB6612 驱动模块都有 16 个引脚，为了驱动 1 个电机，具体的连线规则如下：

### ➡️ 第 1 块：控制【左前电机】
| TB6612丝印 | 连接到何处 | 具体引脚与作用说明 |
|:-|:-|:-|
| **VM** | 电池电源正极 | 纯动力供电 (必接) |
| **VCC** | 单片机 3.3V/5V | 驱动板逻辑供电 (必接) |
| **GND** (3个) | 全部接系统GND | 所有地线互通 (必接) |
| **STBY** | 单片机 `PD0` | 唤醒使能，否则不工作，4个模块均接此引脚 |
| **PWMA** | 单片机 `PA8` | 左前轮的速度控制 (TIM1_CH1) |
| **AIN1** | 单片机 `PC0` | 左前轮的正反转引脚1 |
| **AIN2** | 单片机 `PC1` | 左前轮的正反转引脚2 |
| **AO1** | 左前电机线1 (`M+`) | 如果方向反了，交换AO1和AO2 |
| **AO2** | 左前电机线6 (`M-`) | 见上 |
| **PWMB/BIN1/BIN2** | 悬空不接 | (单模块单电机使用) |
| **BO1/BO2** | 悬空不接 | (单模块单电机使用) |

### ➡️ 第 2 块：控制【右前电机】
| TB6612丝印 | 连接到何处 | 具体引脚与作用说明 |
|:-|:-|:-|
| VM, VCC, GND | 同上 | (必接) |
| **STBY** | 单片机 `PD0` | (必接) |
| **PWMA** | 单片机 `PA9` | 右前轮的速度控制 (TIM1_CH2) |
| **AIN1** | 单片机 `PC2` | 右前轮的正反转引脚1 |
| **AIN2** | 单片机 `PC3` | 右前轮的正反转引脚2 |
| **AO1** | 右前电机线1 (`M+`) | 如果方向反了，交换AO1和AO2 |
| **AO2** | 右前电机线6 (`M-`) | 见上 |
| PWMB~BO2 | 悬空不接 | (单模块单电机使用) |

### ➡️ 第 3 块：控制【左后电机】
| TB6612丝印 | 连接到何处 | 具体引脚与作用说明 |
|:-|:-|:-|
| VM, VCC, GND | 同上 | (必接) |
| **STBY** | 单片机 `PD0` | (必接) |
| **PWMA** | 单片机 `PA10` | 左后轮的速度控制 (TIM1_CH3) |
| **AIN1** | 单片机 `PC4` | 左后轮的正反转引脚1 |
| **AIN2** | 单片机 `PC5` | 左后轮的正反转引脚2 |
| **AO1** | 左后电机线1 (`M+`) | 如果方向反了，交换AO1和AO2 |
| **AO2** | 左后电机线6 (`M-`) | 见上 |
| PWMB~BO2 | 悬空不接 | (单模块单电机使用) |

### ➡️ 第 4 块：控制【右后电机】
| TB6612丝印 | 连接到何处 | 具体引脚与作用说明 |
|:-|:-|:-|
| VM, VCC, GND | 同上 | (必接) |
| **STBY** | 单片机 `PD0` | (必接) |
| **PWMA** | 单片机 `PA11` | 右后轮的速度控制 (TIM1_CH4) |
| **AIN1** | 单片机 `PC6` | 右后轮的正反转引脚1 |
| **AIN2** | 单片机 `PC7` | 右后轮的正反转引脚2 |
| **AO1** | 右后电机线1 (`M+`) | 如果方向反了，交换AO1和AO2 |
| **AO2** | 右后电机线6 (`M-`) | 见上 |
| PWMB~BO2 | 悬空不接 | (单模块单电机使用) |

## 4. OpenMV H7 Plus 视觉模块
使用 USART1，波特率 115200。**注意 RX 接 TX 交叉连线！**

| OpenMV 引脚 | STM32单片机引脚 | 作用 |
| :--- | :--- | :--- |
| **VIN/5V** | 单片机 5V | 供电必须够 |
| **GND** | 任意地线 GND | 必须共地 |
| **P4 (TX)** | 单片机 `PB7` (UART1_RX) | 视觉数据传给单片机 |
| **P5 (RX)** | 单片机 `PB6` (UART1_TX) | 单片机回传给视觉模块 |

---

## 5. 语音合成模块 (YS-V6 / SYN6658)
使用 USART2，波特率通常为 9600 或 115200 (根据你的软件配置)。**同样需交叉连线！**

| SYN6658 模块引脚 | STM32单片机引脚 | 作用 |
| :--- | :--- | :--- |
| **VCC(5V)** | 单片机 5V供电 | 必须供电 |
| **GND** | 任意地线 GND | 必须共地 |
| **TXD** | 单片机 `PA3` (UART2_RX) | 模块发给单片机（可选接） |
| **RXD** | 单片机 `PA2` (UART2_TX) | 单片机把要读的字发给模块（必接）|

| 串口姿态模块引脚 | STM32单片机引脚 | 说明 |
